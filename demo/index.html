Buttons

Like all components of Trowel, buttons have numerous default setting based on your base colors.

Of course these are easily customised. Here are the SASS variables (and default values) available to change.

Variable                        Default Value               Available Options
----------------------------------------------------------------------------------------------------
$button_theme                   is-brand                    Any is-[theme] string
$button_style                   is-rounded                  is-sharp, is-pill, is-rounded
$button_type                    is-flat                     is-flat, is-outlined, is-3D, is-gradient
$button_size                    is-regular                  is-[tiny|small|regular|large|big|huge|jumbo]
$button_padding                 0.36em 1.6em 0.38em
$button_margin                  has-margin-tiny             has-margin-[tiny|small|regular|large|big|huge|jumbo]
$button_case                    is-capitalized              is-capitalized, is-uppercase
$button_font_weight             300
$button_border_width            2px
$button_3D                      false                       | You can set default button styles
$button_outlined                false                       | using these flags. By setting defaults
$button_gradient                false                       | you can clean up the html to the minimal
$button_shadow                  true                        | <button>Label</button>
